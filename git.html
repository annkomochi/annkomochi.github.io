github-pagesの作り方

１.github Pagess用のリポジトリ作成

New　repositoryのPepository name 欄にユーザー名.github.ioを入力し、
Create reppositoryボタンをクリック

2.ローカルにリポジトリをクローンする

ここからはローカルでの作業になります。
ローカルのターミナルに次のように入力します

git clone https://github.com/username/username.github.io

これでリポジトリがローカルにも反映されました。
実際にリポジトリに移動して確認してみましょう

cd username.github.io

移動できましたか？これで作成できました！おめでとうございます。

github-pagesの更新の仕方

htmlファイルを作成します

次にステージングしコミットをします

git add ファイル名

git commit -m "Initial commit"

GitHubへプッシュ

git push -u origin master

更新が完了しました。おめでとうございます。
