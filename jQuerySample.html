自動でスクロールするjQuery

→DEMO

HTML


            <div id="one">
            <p id="scroll">ここをクリック</p>
            </div>
            <div id="two"></div>

jQuery


            $('#scroll').click(function(){
              $("html,body").animate({scrollTop:$('#two').offset().top});
            })

サイトのトップページでよく見かけるスクロールを促すボタンをクリックすると自動でスクロールするあいつです。jQuryを使うとこんなにも簡単に出来る事に驚きました。


                  $("html,body").animate({scrollTop:$('#two').offset().top});

スクロールするとトップへ戻るが現れるjQuery

→DEMO

HTML


            <p id="pagetop"><a href="#one">PAGE TOP</a></p>

これを実装したいページのコードの下の方に張り付けます。

jQuery


            $(function() {
              var topBtn = $('#pagetop');    
              topBtn.hide();
              //スクロールが200に達したらボタン表示
              $(window).scroll(function () {
                  if ($(this).scrollTop() > 200) {
                      topBtn.fadeIn();
                  } else {
                      topBtn.fadeOut();
                  }
              });
          
          });

画像をふわふわさせるjQuery

HTML


              <div id="content">
              <img  id="fly" src="" alt="" >
              </div>

今回は画像でしたが文字とかでも大丈夫です。

css


            #content {
              height: 300px;
            }

親要素に高さを指定しないと下の要素に影響するので注意

jQuery


              $(function () {
                setTimeout('rect()'); //アニメーションを実行
            });
             
            function rect() {
                $('#fly').animate({
                    marginTop: '-=10px'
                }, 800).animate({
                    marginTop: '+=10px'
                }, 800);
                setTimeout('rect()', 1600); //アニメーションを繰り返す間隔
            }
