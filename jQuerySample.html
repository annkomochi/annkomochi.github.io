自動でスクロールするjQuery

→DEMO

HTML


            <div id="one">
            <p id="scroll">ここをクリック</p>
            </div>
            <div id="two"></div>

jQuery


            $('#scroll').click(function(){
              $("html,body").animate({scrollTop:$('#two').offset().top});
            })

サイトのトップページでよく見かけるスクロールを促すボタンをクリックすると自動でスクロールするあいつです。jQuryを使うとこんなにも簡単に出来る事に驚きました。

解説


                  $("html,body").animate({scrollTop:$('#two').offset().top},1000);

scrolletop:のあとに移動したい場所のスクロール数をいれます。
1000ミリ秒（1秒）で移動します。

スクロールするとトップへ戻るが現れるjQuery

→DEMO

HTML


            <p id="pagetop"><a href="#one">PAGE TOP</a></p>

これを実装したいページのコードの下の方に張り付けます。

jQuery


            $(function() {
              var topBtn = $('#pagetop');    
              topBtn.hide();
              //スクロールが200に達したらボタン表示
              $(window).scroll(function () {
                  if ($(this).scrollTop() > 200) {
                      topBtn.fadeIn();
                  } else {
                      topBtn.fadeOut();
                  }
              });
          
          });

解説


                topBtn.fadeIn();

フェードイン


                  topBtn.fadeOut();

フェードアウト、簡単ですね

画像をふわふわさせるjQuery

HTML


              <div id="content">
              <img  id="fly" src="" alt="" >
              </div>

今回は画像でしたが文字とかでも大丈夫です。

css


            #content {
              height: 300px;
            }

親要素に高さを指定しないと下の要素に影響するので注意

jQuery


            $(function () {
              setTimeout('fly()');
            });
          
            function fly() {
              $('#fly').animate({
                marginTop: '-=10px'
              }, 800).animate({
                marginTop: '+=10px'
              }, 800);
              setTimeout('fly()', 1600); //アニメーションを繰り返す間隔
            }

解説


                  marginTop: '-=10px'

ここでマージンを削って浮いたようにします。


                    marginTop: '+=10px'

こんどはマージン増やして沈んだようにします。

→DEMO

jQuery


              $(function() {
                var topBtn = $('#pagetop');    
                topBtn.hide();
                //スクロールが100に達したらボタン表示
                $(window).scroll(function () {
                    if ($(this).scrollTop() > 100) {
                        topBtn.fadeIn();
                    } else {
                        topBtn.fadeOut();
                    }
                });
            
                    //スクロールしてトップ
                    topBtn.click(function () {
                    $('body,html').animate({
                        scrollTop: 0
                    }, 500);
                    return false;
                });
            });

上で紹介した二つのjQueryを使えばアニメーションで上にスクロールしながらトップに戻ることが出来ます。
