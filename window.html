Window.openとは

window.openは、新しくウィンドウを開かせるメソッドです。

サンプルコード

htmlコード

<input type="button" value="window.open サンプル" id="window">

javascriptコード

var windowopn = function(){
var win = window.open('subwin.html','subwin','width=640,height=480');
}

document.getElementById('window').addEventListener('click',windowopn,false);

サンプルプログラム

説明

window.open(url','ウィンドウ名','オプション')

URL 新しく開くウィンドウのURLを指定する
ウィンドウ名 特に指定がない場合nullでも大丈夫です。新しいウィンドウのタイトルではありません。
オプション 新規ウィンドウに様々な指定をすることができる。

オプション一覧(一部)

オプション 値 説明
width 数値 ウィンドウの幅を指定
height 数値 ウィンドウの高さを指定
left 数値 ウィンドウの表示位置　左
top 数値 ウィンドウの表示位置　上
menubar yes/no メニューバーを表示するか
toolbar yes/no ツールバーを表示するか
location yes/no アドレスバーを表示するか
status yes/no ステータスバーを表示するか
resizable yes/no リサイズを許可するか
scrollbars yes/no スクロールバーを表示するか

Window.closeとは

現在のウィンドウを閉じるメソッドです。

サンプルコード

htmlコード

<input type="button" value="閉じる" id="windowclose">

javascriptコード

document.getElementById('windowclose').addEventListener('click',function(){
window.open('about:blank','_self').close();
})

サンプルプログラム


説明

window.close

ウィンドウを閉じるメソッドですが、現在これをただ書くだけでは閉じることは出来ないみたいです。

サンプルコードでは

window.open('about:blank','_self').close();

と言う様に書いています。これでどのブラウザでもウィンドウを閉じることができます。

Window.focus

指定したウィンドウにフォーカスを当てて最前面に表示する。

サンプルコード

htmlコード

<input type="button" value="クリックして新しいWindowを開く"  id="window4">
<input type="button" value="フォーカスを当てる" id="window5>

javascriptコード

var win
// window.openであらかじめウィンドウを開いておく
var windowopn = function(){
win = window.open('subwin.html','subwin','width=640,height=480');
}

// フォーカスをあてる。
var windowfocus = function() {
  win.focus();
}

document.getElementById('window4').addEventListener('click',windowopn,false);
document.getElementById('window5').addEventListener('click',windowfocus,false);

サンプルプログラム

説明

window.focus

この一文だけでフォーカスを当て最前面に表示することができます。
